Input Data: [Enabler Description, Keyword, and Passage]

Evaluate if a Keyword is "significant" or "not significant" relative to a provided Enabler Description based on a 3-sentence Passage.

Output: One word only ("significant" or "not significant").

Criteria for "significant":

The Current sentence (supported by context) must link the Keyword to the Enabler Description.

Criteria for "not significant":

The Passages that do not link the Keyword to the Enabler Description. Passages from papers' references or related works must be considered "non significant".
